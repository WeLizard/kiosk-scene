# demo-generic

This app is a redistribution-safe demo instance for the public `kiosk-scene` core.

Current focus:

- left avatar panel with three view presets
- right carousel shell that mirrors the original Neiri overview composition
- first page weather overview modeled after the current `neiri-scene` layout
- dynamic follow-up pages driven by `scene.json`

Rules:

- no proprietary avatar model files;
- no private Home Assistant entity names;
- no user secrets;
- no deployment-specific branding.
